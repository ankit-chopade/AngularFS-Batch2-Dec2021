Working with Git

--> Git is a distributed Version control System and source code management.

Repositories
~~~~~~~~~~~~
1. Central Repositories		---		Github (internet / remove)
2. Local Repositories			---		local system (with/without internet)





Working  Directory 	--->   Staging Area  / Cache Area  ---> 	Git Directory   --->  	Move to Central


1. Working  Directory
			>   git init
			
			Initialized empty Git repository in D:/gitdemo/.git/
			
			
2, Staging Area  / Cache Area  
			git add  filename
			git add  -A				-----  for all files
			git add  .					-----  for all files
			
			
			git status
			
3. Git Directory
			git commit   -m  "commit message"
			
		Note:	-m  refers  the message
	
4. Move to Central
			git checkout -b main
			git push  origin  main
